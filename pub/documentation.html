CSC309_graphLibrary API

getting started:

To use CSC309_graphLibrary, first include these two lines in html file:

API:

Objects:

Name Type Description
BarChart object

The chart provided by the library that can help both front-end user and developer to make barchart

, either through code or through user interaction.

PieChart object

The chart provided by the library that can help both front-end user and developer to make PieChart,

either through code or through user interaction.

LineChart object

The chart provided by the library that can help both front-end user and developer to make LineChart

, either through code or through user interaction.

Methods:

Name return value Description
addForm none

the method enable graph to have a input bar on top of it, which allows front-end user to interact with the graph.

Piechart.addData none

the method which allow piechart to add data. The first parameter specify the data, which is a integer.

the second parameter is the category, which is a string

Barchart.addData none

the method which allow barchart to add data. The first parameter specify the data, which is a integer.

the second parameter is the category, which is a string

LineChart.addData none

the method which allow linechart to add data. first parameter is the x value, second parameter is the y value.

render none

the method that is responsible for showing the graph on screen.
